Registrapp es una aplicacion realizada para poder registrar las asistencias de los estudiantes los cuales asisten a sus clases en el Duoc UC

    Registro

    En este pagina de la app se puede realizar el registro de los alumnos para que esto quede guardado en una base de datos

  • 02
    Inicio de Sesion

    Se puede realizar uin inicio de sesion de forma efectiva haciendo la comprobacion con la base de dato

  • 03
    Recuperar

    Se puede realizar una recuperacion de contraseña con el correo electronico

  • 04
    Usuario

    Se pueden realizar los registros de los alumnos a cada clase

    QR
